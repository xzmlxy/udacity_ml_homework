def performance_metric2(y_true, y_predict):
    """"""
    array_true = np.array(y_true)
    array_pred = np.array(y_predict)
    var = array_true.var()
    array_temp = array_true-array_pred
    sse = np.square(array_temp).sum()
    score = 1-sse/var/len(y_true)
    return score

问题 7- 网格搜索（Grid Search）¶

问题 7 - 回答:网格搜索法就是指将模型的参数指定为多个可选项，那么当存在多个参数时，就形成了一个参数取值的多维网格。它可以很方便的测试多种不同的参数组合，并返回最佳的参数选择，从而达到优化模型的效果。¶

问题 8 - 交叉验证¶

  • Kk-fold cross-validation
  • GridSearchCV
  • GridSearchCV'cv_results_'

fit_model print pd.DataFrame(grid.cv_results_)

问题 8 - 回答：（1）K折交叉验证法就是将数据平均分为K份。将其中的K-1份作为训练数据，而1份作为测试数据进行模型训练和评估。不断更换测试数据，直至完成K次，最后将这K次实验的结果进行平均，作为最终的结果。这能高效的利用数据，并能得到更为准确的模型评估。（2）在使用GridSearchCV时，可以通过赋值参数cv来指定具体的交叉验证类型，如果不做设置，那么默认的是K=3的K折交叉验证。（3）GridSearchCV中的cvresult属性以表格的形式，返回了所有的参数组合以及在不同参数组合下的得分、时间等等指标。（4）如果不用交叉验证，我觉得可能的问题是：仅使用固定的训练数据做一次评估，得到的最优参数可能是不够准确的。我的fit_model函数返回的最佳值是4，但打印grid.cvresults来看，并不是每次训练的最佳结果都是4。而是经过K次试验后，参数取4时的平均得分最高，这有利于规避偶然性，因此我觉得交叉验证能提高参数选择的准确性。¶
